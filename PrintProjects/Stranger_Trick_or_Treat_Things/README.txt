# Summary

A Halloween mash-up of 'Trick or Treat' and the 'Stranger Things' logo. Designed using the same font, ITC Benguiat, but altered to make it 3D printable.

Have also uploaded a flat version which you can use to make a light box (print upside down, no infill and no top) and artwork in various formats for your remixing pleasure.
