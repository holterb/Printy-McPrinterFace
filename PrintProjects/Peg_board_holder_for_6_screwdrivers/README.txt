# Summary

This is a holder for 6 small screwdrivers. You can find these everywhere in hardware stores. They normally come in boxes of 6 with different sizes.This holder fits to a standard peg board with 1 inch spacing. The OpenSCAD file is included so you can change the design if you want.

Other pegboard tools which I designed can be found at:
Universal box http://www.thingiverse.com/thing:1241017
Multimeter holder http://www.thingiverse.com/thing:2024376

For a table top screwdriver holder see my design: https://www.thingiverse.com/thing:2725510
