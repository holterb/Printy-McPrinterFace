# Summary

This is a simple ribbed vase meant to be printed as one continuous spiral.
It can be used for flowers, snacks or as standalone decoration.

"ribbed_vase.stl" is the base model, "ribbed_vase_tall.stl" is just a minor rescale of it to make it more fitting for flowers.

I am always happy to see your makes!

# Print Settings

Printer Brand: Creality
Printer: CR-10S
Rafts: No
Supports: No
Resolution: Doesn't Matter
Infill: No

Notes: 
Print using spiral or "vase" mode.
Make sure to use 3+ bottom layers and overextrude a bit to stop it from leaking water.
