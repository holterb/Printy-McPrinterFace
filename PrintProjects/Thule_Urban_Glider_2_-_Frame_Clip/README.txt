# Summary

This is custom frame clip for cup holder I made for Thule Urban Glider 2 stroller. I saw that some Thule frames are very similar so it might fit another models. 

I printed it with three walls and 20% infill so it is tight fit. I also tried to print it with two walls and 10% infill and it was a little bit more flexible but it worked too. 

Brim recommended, support needed for the peg.
